# Expense Tracker

Expense Tracker is a simple Angular application for managing personal expenses.
It allows users to add expenses, filter them by category and amount, and view the total cost.
---

## Features

- Add expenses with amount, category, and date
- Filter expenses by category
- Search expenses by minimum and maximum amount
- Automatically calculated total expenses
- Responsive and clean user interface

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/KroshneiRostyslav/Oblik_vytrat.git

2. Navigate to the project directory:
  cd Oblik_vytrat

3. Install dependencies:
  npm install

4. Start the development server:
  ng serve

5. Open the application in your browser:
  http://localhost:4200
